# In VS Code

Install the [Mermaid Preview](https://marketplace.visualstudio.com/items?itemName=vstirbu.vscode-mermaid-preview) extension.

To view the diagram while editing the `.mmd` files:

- Open a file containing Mermaid diagram
- Choose Preview Mermaid Diagram from your Command Pallette
- Move cursor inside the diagram
